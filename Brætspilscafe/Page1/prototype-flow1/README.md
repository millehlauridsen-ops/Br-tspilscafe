Prototype — Flow 1

Indhold:
- `index.html` — entré-HTML med tre screens (liste, detaljer, booking)
- `styles.css` — grundlæggende styling, mobilramme 390x844
- `app.js` — simpel navigationslogik med data-next / data-prev
- `assets/` — SVG placeholder assets i de korrekte mål
 - `assets/` — SVG placeholder assets i de korrekte mål (erstat eller tilføj PNG frames)

Placer de vedhæftede PNG-filer i `Page1/prototype-flow1/assets/` med følgende navne så prototypen bruger dem automatisk:
 - `uno-cover.png` (kort cover/thumbnail til featured)
 - `codenames-thumb.png` (thumbnail for Codenames)
 - `azul-thumb.png` (thumbnail for Azul)
 - `exploding-thumb.png` (thumbnail for Exploding Kittens)
 - `uno-detail.png` (stor detail-frame PNG)

Når filerne er lagt i mappen, åbn `Page1/prototype-flow1/index.html` i browser for at se prototypen med dine frames.

Hvordan køre:
Åbn `Page1/prototype-flow1/index.html` i din browser (dobbeltklik eller brug Live Server i VS Code).

Note:
Jeg har brugt dimensioner fundet i din eksisterende `Page1` (phone 390×844, featured 357×209, card 124×124). Hvis du ønsker præcise Figma-dimensionsoplysninger fra frames i filen, kan jeg hente dem via Figma API hvis du giver en personal access token.
